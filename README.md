# Upgrade_python
In this repo, I document my Python learning journey. 
My goal is to strengthen my understanding of core Python concepts and real-world applications in Data Engineering.
They include exercises, mini-projects, and database-related practice, with emphasis on developing ETL/ELT pipeline skills. 

## 📚 1. Project Structure

- `exercise/` → Practice problems and small scripts
- `python_drill/` → Jupyter notebooks for concept drills
- `databases/` → Database-related practice
- `db_con.py` → Database connection experiments


## 📚 2. Topics Covered

- Variables & Data Types
- Control Flow (if, loops)
- Functions
- Lists & Dictionaries
- File Handling
- Database Connections (basic)
